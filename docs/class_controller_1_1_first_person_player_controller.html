Detailed Description

First-person player controller for movement, looking, and basic interaction checks. Uses a capsule cast to block movement and slide along walls. Rotation is smoothed to reduce jitter. Pitch can be enabled for the camera. Also scans a small radius for interactable objects and updates a list. Can read speed from a Stats component if available.
