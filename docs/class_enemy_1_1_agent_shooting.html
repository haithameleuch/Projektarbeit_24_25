Detailed Description

Defines a ranged enemy agent (AgentShooting) trained with Unity ML-Agents.

Core Behavior:

  • Acts like a stationary turret that rotates and fires projectiles.
  • Receives continuous actions for Y-axis rotation and a discrete action for firing.
  • Uses projectile pooling via ObjectPoolManager for performance.

Observations:

  • Targetâ€™s normalized health (float)
  • Direction to target in local space (Vector3)
  • Dot product of forward vs. target direction (float)
  • Angular difference to target, normalized [0,1] (float)

Total = 6 observations per step.

Rewards:

  • Positive reward for aiming closer to the player (dot product alignment).
  • Large positive reward for killing the player (health = 0).
  • Small negative time penalty each step to promote efficiency.

Key Features:

  • Freezes Rigidbody to prevent physics-based movement.
  • Keeps enemy model visually aligned with the gun barrel (shootPoint).
  • Supports coroutine-based initialization to wait for player spawn.
  • Uses cooldown-based firing logic to prevent spamming.

Best suited for:

  • Turret-style enemies.
  • Ranged AI requiring line-of-sight targeting.
  • Training scenarios where accuracy and timing are critical.
