Detailed Description

Defines a navigation-focused enemy agent (HunterAgentObstacle) trained with Unity ML-Agents.

Core Behavior:

  • Chases the player while avoiding obstacles using continuous forward and rotation actions.
  • Receives rewards for reducing distance, aligning movement direction, and facing the target.
  • Penalized for getting stuck or colliding with obstacles, walls, or doors.

Observations:

  • Agent's local position (Vector3)
  • Target's local position (Vector3)
  • Normalized vector from agent to target (Vector3)
  • Agent's normalized velocity (Vector3)
  • Agent's forward vector (Vector3)

Total = 15 observations per step.

Rewards:

  • Positive reward for reducing distance to the player (progress reward).
  • Positive reward for aligning movement direction with the target (velocity alignment).
  • Positive reward for facing the target directly (facing reward).
  • Negative reward for being stuck or colliding with obstacles.

Key Features:

  • Physics-based movement with Rigidbody and smooth rotation.
  • Stuck detection using position deltas and a timer threshold.
  • Coroutine-based player detection for dynamic spawns.
  • Gizmos for debugging forward direction and target tracking.

Best suited for:

  • Obstacle-aware navigation tasks.
  • Precision pursuit and pathfinding AI training.
  • Navigation reward shaping scenarios.
