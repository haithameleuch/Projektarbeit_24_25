Detailed Description

Defines a pursuit-type enemy agent (HunterAgent) trained with Unity ML-Agents.

Core Behavior:

  • Moves toward the player using continuous forward movement and rotation actions.
  • Receives rewards for reducing distance to the player and properly facing the target.
  • Penalized for being stuck, hitting walls, or inefficient movement.

Observations:

  • Normalized vector to the player (Vector3)
  • Agent's forward vector (Vector3)
  • Signed angle between forward and target direction (float, normalized)

Total = 7 observations per step.

Rewards:

  • Positive reward for decreasing distance to the player.
  • Positive reward for facing the player directly (alignment).
  • Negative rewards for being stuck or colliding with walls.
  • Step penalty to encourage efficient navigation.

Key Features:

  • Rigidbody-based physics movement with smooth rotation.
  • Stuck detection using positional deltas and timers.
  • Coroutine-based player detection to handle dynamic spawns.
  • Modular design suitable as a base for advanced enemy AI types.

Best suited for:

  • Pursuer enemies that chase players directly.
  • Training scenarios focusing on obstacle-aware navigation and directional control.
