Detailed Description

Defines an enemy agent (BomberAgent) trained with Unity ML-Agents.

Core Behavior:

  • Pursues the player using continuous movement and rotation actions.
  • Drops bombs when in proximity to the player using a coroutine-based system.
  • Receives rewards for approaching and facing the player accurately.
  • Penalized for collisions with walls or being stuck.

Observations:

  • Normalized direction to the player (Vector3)
  • Agent’s forward direction (Vector3)
  • Signed angle to the player (float, normalized)

Total = 7 observations per step.

Rewards:

  • Positive reward for reducing distance to the player.
  • Positive reward for facing the player (alignment).
  • Large positive reward for touching the player (end of episode).
  • Negative rewards for being stuck or colliding with walls.

Key Features:

  • Rigidbody-based physics movement and rotation.
  • Coroutine-based bomb-dropping with cooldown and proximity checks.
  • Initialization waits for player spawn to ensure proper targeting.
  • Stuck detection and distance-based reward shaping.

Best suited for:

  • Pursuer-type enemies with area-of-effect attacks.
  • Training scenarios emphasizing navigation, threat generation, and reward shaping.
