Detailed Description

Defines a stealthy and agile enemy agent (GhostAgent) trained with Unity ML-Agents.

Core Behavior:

  • Moves smoothly toward the player using continuous movement and rotation actions.
  • Performs dashes to the left or right using Vector3.Lerp for smooth interpolation.
  • Becomes temporarily invisible by adjusting the Renderer material alpha.
  • Receives rewards for approaching and facing the player accurately.
  • Penalized for hitting walls, getting stuck, or poor performance.

Observations:

  • Direction to the player (Vector3)
  • Normalized agent forward vector (Vector3)
  • Signed angle between forward direction and player (float, normalized)

Total = 7 observations per step.

Rewards:

  • Positive reward for reducing distance to the player.
  • Positive reward for facing the player (alignment).
  • Negative rewards for being idle, stuck, or colliding with walls.

Key Features:

  • Rigidbody-based physics movement for realistic motion.
  • Smooth dash movement via Vector3.Lerp over dashDuration.
  • Visibility control using material alpha fading for stealth effect.
  • Randomized dash routines to increase unpredictability.

Best suited for:

  • Stealth and agile enemy types.
  • Scenarios emphasizing unpredictable movement, evasion, and player pursuit.
